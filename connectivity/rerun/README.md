# Rerun

This example streams a pointcloud to [Rerun Viewer](https://www.rerun.io/)

# Installation

```
python3 -m pip install -r requirements.txt
```

## Usage

Run the application

```
python3 main.py
```

You should see a rerun window pop up with a pointcloud displayed within:
