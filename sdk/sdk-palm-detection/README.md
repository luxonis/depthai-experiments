Palm detection
================

This example demonstrates
[palm detection network](https://google.github.io/mediapipe/solutions/hands#palm-detection-model) using DepthAI SDK.

## Pre-requisites

```bash
python3 -m pip install -r requirements.txt
```


## Usage

